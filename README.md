## 개요
이 프로젝트는 react tutorial 에 사용된 소스입니다.  
사용된 Bootstrap CSS framework는 reactstrap 입니다..
라우팅은 react-router-dom을 적용하였습니다.
그외에,
페이지네이션을 하기위해서 rc-pagination을 적용하였고,
토스트를 보여주기 위해서 react-toast-notifications 가 사용되었습니다.
